# 🖨️ نظام حساب تكلفة طباعة الاستيكرات

نظام احترافي لحساب تكلفة طباعة الاستيكرات مع واجهة عربية كاملة ودعم RTL.

## ✨ المميزات

### 📋 إدخال شامل للبيانات
- بيانات العميل (اسم، رقم، مندوب)
- مواصفات الاستيكر (المقاس، الكمية، الشكل)
- خيارات الطباعة (4-7 ألوان)
- التشطيبات (سلفان، بصمة، سلك سكرين، سبوت UV)
- نوع التركيب واتجاه الرول

### 💰 حساب تلقائي فوري
- حساب المساحة الإجمالية
- حساب سعر الوحدة
- حساب الضريبة (15%)
- الإجمالي النهائي شامل الضريبة

### 🎨 تصميم عصري واحترافي
- لوحة ألوان مريحة للعين
- تباين عالي للوضوح
- تصميم متجاوب لجميع الشاشات
- أنيميشن سلس
- أيقونات توضيحية

### ⚙️ قابل للتخصيص
- جميع الأسعار في ملف منفصل (`/config/pricingConfig.ts`)
- سهل التعديل والتحديث
- معادلات واضحة ومفهومة

## 🚀 التقنيات المستخدمة

- **React** - مكتبة بناء الواجهات
- **TypeScript** - لغة البرمجة المستخدمة
- **Tailwind CSS** - إطار عمل التصميم
- **Lucide Icons** - مكتبة الأيقونات

## 🔧 تعديل الأسعار

لتعديل الأسعار، افتح ملف `/config/pricingConfig.ts` وقم بتغيير القيم:

```typescript
export const PRICING_CONFIG = {
  basePricePerSqm: 50,        // السعر الأساسي للمتر المربع
  printTypes: {
    'cmyk-4': 0,              // CMYK قياسي
    'colors-5': 15,           // 5 ألوان
    'colors-6': 25,           // 6 ألوان
    'colors-7': 35,           // 7 ألوان
  },
  // ... المزيد من الخيارات
};
```

## 📱 الاستخدام

1. **أدخل بيانات العميل** - املأ اسم العميل ورقمه ومعلومات المندوب
2. **حدد مواصفات الاستيكر** - اختر المقاس، الكمية، الشكل، نوع الطباعة
3. **اختر التشطيبات** - فعّل الخيارات الإضافية حسب الحاجة
4. **شاهد السعر مباشرة** - يتحدث الملخص تلقائياً
5. **إرسال العرض** - اضغط على زر "إرسال العرض للعميل"

## ⚡ ملاحظات مهمة

- **لا يوجد Backend حالياً** - جميع الحسابات تتم في المتصفح
- **البيانات غير مخزنة** - لا توجد قواعد بيانات
- **للاستخدام الداخلي** - مناسب للاستخدام في مكاتب الطباعة
